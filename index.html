Keliling Segitiga

Rumus keliling segitiga:

K = a + b + c

Keterangan:

K = Keliling

a = Sisi

b = Sisi

c = Sisi






Luas Segitiga

Rumus luas segitiga:

L = 1/2 x a x t

Keterangan:

L = Luas

a = Panjang Alas

t = Tinggi
